# 八字 + 车牌选号工作流

## 触发场景

用户想按生辰八字、五行、民俗谐音或易记性比较候选车牌时使用本工作流。

## 输入检查

必须有：
- 公历出生日期
- 性别，允许未知
- 候选车牌列表

出生时间可以未知。未知时辰会退回简化五行参考，并在输出里标低置信度。

可选：
- 出生地
- 当前省份或城市前缀，例如 `粤` 或 `粤B`
- 自定义规则文件

## 调用方式

```bash
python scripts/license_plate_recommend.py \
  --birth-date 1990-05-12 \
  --birth-time 08:30 \
  --gender female \
  --plates 粤B8A68Z 粤B52088 粤B3M96Q
```

如果已安装为 Python 包，也可以运行：

```bash
python -m license_plate_bazi.cli \
  --birth-date 1990-05-12 \
  --birth-time 08:30 \
  --gender female \
  --plates 粤B8A68Z 粤B52088 粤B3M96Q
```

## 评分结构

总分 100 分：
- 八字喜用神匹配：40
- 数字组合吉利度：25
- 字母组合匹配：15
- 整体易记/美观：10
- 谐音和避雷项：10

规则文件：`config/license_plate_rules.json`。

## 解释原则

说明要落到具体字符和规则：
- 哪些字符属于喜用五行
- 哪些字符落在避重五行
- 数字是否有重复、顺子、回文、常见谐音
- 字母是否补足五行
- 当前地区前缀是否匹配

不要绝对化，不要把车牌和财富、疾病、事故等结果绑定。

# 八字车牌推荐 Demo

## 输入样例

- 出生日期：1990-05-12
- 出生时间：08:30
- 性别：female
- 候选车牌：`粤B8A68Z`、`粤B52088`、`粤B3M96Q`

## 命令行运行方式

先安装主项目排盘依赖：

```bash
cd calculator
npm install
cd ..
```

再运行车牌推荐：

```bash
python3 scripts/license_plate_recommend.py \
  --birth-date 1990-05-12 \
  --birth-time 08:30 \
  --gender female \
  --plates 粤B8A68Z 粤B52088 粤B3M96Q
```

## 输出样例摘要

完整输出是 Markdown，核心排名如下：

| 排名 | 车牌 | 综合分 | 推荐等级 | 八字匹配 | 数字组合 | 字母匹配 | 易记美观 | 谐音避雷 |
|---:|---|---:|---|---:|---:|---:|---:|---:|
| 1 | 粤B8A68Z | 88/100 | 强推荐 | 40.0/40 | 16.7/25 | 15.0/15 | 6.5/10 | 10.0/10 |
| 2 | 粤B52088 | 79/100 | 可选 | 40.0/40 | 17.3/25 | 6.0/15 | 5.5/10 | 10.0/10 |
| 3 | 粤B3M96Q | 74/100 | 可选 | 32.0/40 | 12.5/25 | 15.0/15 | 5.0/10 | 10.0/10 |

## 为什么第一名更推荐

`粤B8A68Z` 的数字 `8、6、8` 分别映射到土、金、土，字母 `A、Z` 映射到木、金。按本次排盘得到的喜用参考，木、金、水、土都属于较匹配的方向，所以八字匹配和字母匹配分都比较高。数字部分 `868` 还有回文结构，比较容易记。

## 哪些只是一般或可选

`粤B52088` 的数字组合更顺口，也有 `520` 和重复 `88` 的记忆点，但没有后缀字母，字母五行分按保守分处理。

`粤B3M96Q` 的字母匹配不错，但数字里有 `9=火`，本次规则把火列为避重参考，因此存在一个冲突点，整体分低于第一名。

## 输出样例片段

```markdown
### 第1名：粤B8A68Z
综合分：88/100
推荐等级：强推荐

推荐理由：
- 八字五行偏好：喜用参考为木、金、水、土；避重参考为火。
- 数字五行：8=土, 6=金, 8=土。
- 字母五行：A=木, Z=金。
- 数字部分回文，对称感较好。
- 未发现配置中的明显避雷谐音组合。
```
